# Contributing

Thanks for your interest in Skimmy!

**This is a source-available, not open-source, project.** The source is published
so you can read it, audit it, and file precise bug reports — not for reuse,
redistribution, or re-publication. See [`LICENSE`](LICENSE) for the full terms.

## What you *can* do

- **Report a bug.** Open an [issue](https://github.com/giulioc84/skimmy_markdown/issues)
  with reproduction steps, macOS version, and any relevant console output.
- **Suggest a feature.** Open an issue tagged `enhancement`. Describe the
  problem you're trying to solve more than the solution you'd like.
- **Ask a question.** Open a [discussion](https://github.com/giulioc84/skimmy_markdown/discussions)
  if enabled, or an issue otherwise.

## What you *shouldn't* do

- **Please don't send pull requests.** This is a personal project and I can't
  merge contributions without assigning copyright / a CLA, which I haven't set
  up and don't intend to. PRs will be closed with a friendly pointer to this
  document. No hard feelings — just file an issue instead and I'll consider the
  change.
- **Please don't fork the repo to publish your own version** (whether free,
  paid, App Store, or elsewhere). That's what the license forbids.

## Security

If you find a security issue (something that could put other users at risk),
please **don't** open a public issue. Email me at giulio.colla@quietcrafts.nl
with details, and I'll respond as soon as I can.

Thanks for respecting the project's boundaries.
